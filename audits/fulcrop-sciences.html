Quick Wins - Immediate Actions

1
Fix reCAPTCHA Multiple Loading

This single fix could cut 2-3 seconds off load time. Check WPForms settings and any other plugins using reCAPTCHA. Only load on pages with forms.

2
Install Image Optimization Plugin

Use ShortPixel, Smush, or Imagify to automatically convert images to WebP and compress them. Expected savings: 50-70% of image weight.

3
Fix CLS from Popup Modal

Reserve space for the popup or delay its appearance until after page load. Add width/height attributes to all images to prevent layout shifts.

4
Enable WP Rocket or LiteSpeed Cache

Install a caching plugin to serve static HTML, minify CSS/JS, and defer render-blocking resources. This alone can improve scores by 15-25 points.

5
Lazy Load Below-Fold Images

Add loading="lazy" to images not visible on initial page load. Most caching plugins do this automatically.
