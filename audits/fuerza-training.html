Priority Issues

  • 8.5MB Animated GIF Destroying Page Speed
    The homepage hero contains an 8.5MB GIF file (gif_homepage_v3.gif) that takes forever to load. This single file is responsible for most performance problems.
    FIX: Convert the GIF to MP4/WebM video format. This could reduce file size by 90%+ and dramatically improve load times. Or use a static image with CSS animations.
  • Time to Interactive: 58.2 seconds on Mobile
    Users on mobile devices must wait nearly a full minute before they can interact with the page. Most will leave within 3 seconds.
    FIX: Remove/optimize the GIF, defer non-critical JavaScript, and reduce main thread work.
  • 741 KB of Unused JavaScript
    Squarespace is loading commerce and other scripts that may not be needed on every page, blocking rendering.
    FIX: Review Squarespace settings to disable unused features. Consider custom code injection to defer scripts.
  • 78 KB of Unused CSS
    Large CSS file with 95% unused rules is slowing down page rendering.
    FIX: This is a Squarespace limitation. Focus on other optimizations first.
  • Missing Location Keywords in Title
    Title tag doesn't include "Queens" or "Ridgewood" - missing local SEO opportunity.
    FIX: Update title to "Fuerza Training Systems | Gym in Ridgewood, Queens NY"
  • Cart Button Accessibility Issue
    Cart link has mismatched accessible name - says "0 items in cart" but displays "CART 0".
    FIX: Update aria-label to match visible text for screen reader users.
