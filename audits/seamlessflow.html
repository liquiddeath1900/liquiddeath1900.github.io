Critical Issues & Security Vulnerabilities

Missing Security Headers High

Site is missing critical HTTP security headers:

Missing: Content-Security-Policy, Strict-Transport-Security (HSTS), X-Frame-Options, X-XSS-Protection, X-Content-Type-Options

Impact: Vulnerable to clickjacking, XSS attacks, and MIME-type sniffing.

Fix: Use Cloudflare (free) as a proxy to add security headers, or migrate to Netlify/Vercel which support _headers file.

Overly Permissive CORS Policy High

access-control-allow-origin: *

Impact: Any website can make requests to your resources.

Fix: GitHub Pages default. Use Cloudflare to override with stricter policy if needed.

Tailwind CSS CDN in Production Medium

Console: cdn.tailwindcss.com should not be used in production

Impact: Slower load times, dependency on external CDN, not recommended by Tailwind.

Fix: Build Tailwind locally: npx tailwindcss -i input.css -o output.css --minify

Google Analytics Without Cookie Consent Medium

_ga, _ga_B68C3RNBVD cookies set without consent banner

Impact: GDPR/CCPA compliance issue.

Fix: Add cookie consent banner or switch to privacy-friendly analytics (Plausible, Fathom).

Deprecated Meta Tag Low

<meta name="apple-mobile-web-app-capable"> is deprecated
Fix: Replace with: <meta name="mobile-web-app-capable" content="yes">
