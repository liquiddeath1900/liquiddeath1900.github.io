Critical Issues

These issues should be addressed to improve performance and SEO.

1. Tailwind CSS CDN in Production

[WARNING] cdn.tailwindcss.com should not be used in production. To use Tailwind CSS in production, install it as a PostCSS plugin or use the Tailwind CLI

Impact: Slower page load, larger CSS bundle, potential render blocking

Fix: Compile Tailwind CSS locally and serve minified production build. This can reduce CSS size by 90%+

2. Deprecated Meta Tag

<meta name="apple-mobile-web-app-capable" content="yes"> is deprecated

Impact: Minor - may affect iOS PWA behavior in future

Fix: Replace with <meta name="mobile-web-app-capable" content="yes">

3. Missing LocalBusiness Schema Markup

No structured data for LocalBusiness, Organization, or Service schema detected.

Impact: Missing rich snippets in search results, reduced local SEO signals

Fix: Add JSON-LD schema for LocalBusiness with NAP, services, and service areas

4. Hero Video May Impact LCP

Auto-playing video on homepage can slow initial page load on mobile.

Impact: Potential Core Web Vitals issues (LCP)

Fix: Use poster image, lazy-load video, or load after page interaction
